Png transparency creator tool What is a png transparency creator?
This tool replaces any solid color and its close color tones in a Portable Network Graphics (PNG) photo or picture with transparent pixels. To match similar color tones, enter the similarity percentage in the settings. By default, 30% similar color tones are matched but you can change it as you see fit. For example, if the color that you're removing is red, then it will also remove cherry-red, rose-red, wine-red and similar shades of red. Simple and powerful!
